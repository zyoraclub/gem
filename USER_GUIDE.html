1 Installation - Unzip & Setup

Step 1.1: Unzip the Package

Extract the gem-automation.zip file to any folder on your computer.

  • Windows: Right-click the zip file → "Extract All" → Choose a location
  • Mac: Double-click the zip file to extract automatically

Step 1.2: Open Terminal/Command Prompt

Windows:

  1. Open the extracted folder
  2. Hold Shift and right-click inside the folder
  3. Select "Open PowerShell window here" or "Open command window here"

Mac:

  1. Open Terminal (press Cmd + Space, type "Terminal", press Enter)
  2. Type cd (with a space), then drag the extracted folder into Terminal
  3. Press Enter

Step 1.3: Run the Setup Script

This installs all requirements (only needed once):

# Windows:
setup.bat

# Mac:
chmod +x setup.sh && ./setup.sh

Wait for it to complete. This may take 5-10 minutes.

✅ Setup Complete When You See:

"Setup completed successfully! Run start.bat to launch the application"

2 Starting the Application

Step 2.1: Run the Start Script

# Windows:
start.bat

# Mac:
./start.sh

Step 2.2: Open the Dashboard

After the servers start, your browser will automatically open to:

http://localhost:3000

If it doesn't open automatically, manually type this address in your browser.

⚠️ Keep Terminal Open

Don't close the terminal window! The application runs from there. To stop the app, press Ctrl+C in the terminal.

3 Create Gmail App Password

App Password is needed for OTP fetching during GEM form filling.

Step 3.1: Enable 2-Factor Authentication

  1. Go to myaccount.google.com/security
  2. Sign in to your Google account
  3. Under "How you sign in to Google", click "2-Step Verification"
  4. Follow the steps to enable it (use your phone number)

Step 3.2: Generate App Password

  1. Go to myaccount.google.com/apppasswords
  2. Sign in if asked
  3. In "App name" field, type: GEM Automation
  4. Click "Create"
  5. You will see a 16-character password like: abcd efgh ijkl mnop

⚠️ Important!

  • Copy this password immediately - you won't see it again!
  • Remove spaces when entering: abcdefghijklmnop
  • Store it safely - you'll need it for the next step

4 Connect Google Sheets

Step 4.1: Open Integrations Page

  1. In the dashboard, click "Integrations" in the left sidebar
  2. Find the "Google Sheets" card

Step 4.2: Connect Your Account

  1. Click the "Connect Google Sheets" button
  2. A Google login popup will appear
  3. Select your Google account
  4. Click "Continue" (even if it shows "unverified app" warning)
  5. Check all permission boxes and click "Continue"

Step 4.3: Select Your Sheet

  1. After connecting, click "Select Sheet"
  2. Choose your GEM products sheet from the list
  3. The sheet ID will be saved automatically

✅ Connection Successful

You'll see a green "Connected" status with your email displayed.

Your Google Sheet should have these columns:

ColumnContentRequired
AProduct URL (GEM portal link)Yes
BProduct ID (auto-filled by scraper)Auto
EPrice (auto-filled by scraper)Auto
FStatus (auto-filled)Auto
GProduct Code (auto-filled)Auto
KLast Updated timestampAuto
LPrevious PriceAuto
MPrice Changed (YES/NO)Auto

5 Configure Gmail IMAP Settings

This is for reading OTP emails during GEM form automation.

Step 5.1: Enable IMAP in Gmail

  1. Open Gmail in your browser
  2. Click the gear icon ⚙️ → "See all settings"
  3. Go to "Forwarding and POP/IMAP" tab
  4. Under "IMAP access", select "Enable IMAP"
  5. Click "Save Changes"

Step 5.2: Enter Settings in Dashboard

  1. In the dashboard, go to "Integrations"
  2. Find "Gmail IMAP Settings" section
  3. Enter the following:
FieldValue
IMAP Serverimap.gmail.com
Port993
EmailYour Gmail address (same as GEM login)
App PasswordThe 16-character password from Step 3

Step 5.3: Test Connection

  1. Click "Save Settings"
  2. Click "Test Connection"
  3. Wait for success message

✅ IMAP Connected

You'll see "Connection successful! Found X emails in inbox"

6 How to Scrape Products

Step 6.1: Prepare Your Sheet

  1. Open your Google Sheet
  2. In column A, paste GEM product URLs (one per row)
  3. Example URL: https://gem.gov.in/product/12345678

Step 6.2: Go to Dashboard

  1. Click "Dashboard" in the sidebar (or go to home)
  2. You'll see your stats: Total Products, Success Rate, etc.

Step 6.3: Start Scraping

  1. Click the "Start Scraping" button
  2. Wait for the process to complete
  3. Progress will be shown on screen

Step 6.4: View Results

  1. Check your Google Sheet - columns B, E, F, G, K, L, M will be filled
  2. Column M shows "YES" if price changed from last scrape
  3. Dashboard shows updated statistics
  • Product ID: Unique GEM product identifier
  • Price: Current listed price
  • Status: Active, Inactive, etc.
  • Product Code: GEM catalog code
  • Price Tracking: Compares with previous price and marks changes

7 How to Fill Forms (Automation)

📌 Prerequisites

  • Chrome browser installed
  • Gmail IMAP configured (for OTP)
  • Logged into GEM portal in Chrome

Step 7.1: Go to Automation Page

  1. Click "Automation" in the sidebar
  2. Click "Check Chrome" to verify Chrome is detected

Step 7.2: Load Products

  1. Click "Load Data" button
  2. Products from your sheet will appear in the table
  3. Products marked "NEW-PRODUCT" in Status column will be shown

Step 7.3: Select Products

  1. Check the boxes next to products you want to process
  2. Or use "Select All" to select everything

Step 7.4: Start Automation

  1. Click "Start Automation"
  2. Chrome will open and start filling forms automatically
  3. OTPs will be read from your email automatically
  4. Don't touch the Chrome window while running!

Step 7.5: Monitor Progress

  1. Watch the progress on the Automation page
  2. Each product status will update: Processing → Completed/Failed
  3. Log messages show what's happening

✅ Automation Complete

Products will be marked as completed. Check GEM portal to verify uploads.

8 Troubleshooting

Another program is using the port. Fix:

# Windows:
netstat -ano | findstr :8000
taskkill /PID [number] /F

# Mac:
lsof -i :8000
kill -9 [PID number]

Then run start.bat/start.sh again.

  • Make sure you allowed all permissions in Google popup
  • Try disconnecting and reconnecting
  • Check if popup was blocked by browser
  • Clear browser cache and try again
  • Verify App Password is correct (no spaces)
  • Make sure IMAP is enabled in Gmail settings
  • Use your full email address (including @gmail.com)
  • Try generating a new App Password
  • Make sure Chrome is installed
  • Restart the application
  • Windows: Chrome should be in default location
  • Mac: Chrome should be in Applications folder
  • Check if GEM portal website is accessible
  • Verify URLs in column A are correct GEM product links
  • Check your internet connection
  • Some products may be removed from GEM - check manually
  • Verify Gmail IMAP settings are saved correctly
  • Test the IMAP connection in Integrations page
  • Make sure OTP emails are going to inbox (not spam)
  • Check if email address matches GEM account

📞 Need More Help?

If you're still having issues:

  2. Restart the application (Ctrl+C, then run start script again)
  3. Take a screenshot of the error and contact support
